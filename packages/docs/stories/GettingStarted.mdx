# Getting Started

Welcome to the ComponentUI component library! This guide will help you integrate our components into your React application.

## Installation

Install the package using your preferred package manager:

```bash
# npm
npm install @componentui/ui

# pnpm
pnpm add @componentui/ui

# yarn
yarn add @componentui/ui
```

## Setup

### 1. Import Styles

Import the CSS file in your application's entry point:

```tsx
// main.tsx or App.tsx
import '@componentui/ui/styles.css'
```

### 2. Configure Tailwind CSS (Optional)

If you're using Tailwind CSS in your project, you can extend our configuration:

```js
// tailwind.config.js
module.exports = {
  content: [
    './src/**/*.{js,ts,jsx,tsx}',
    './node_modules/@componentui/ui/dist/**/*.js',
  ],
  theme: {
    extend: {
      // Your custom theme
    },
  },
}
```

## Basic Usage

Import and use components in your React application:

```tsx
import { Button, Input, Card } from '@componentui/ui'

function MyApp() {
  return (
    <div>
      <Card>
        <Card.Header>
          <h2>Login</h2>
        </Card.Header>
        <Card.Body>
          <Input
            type="email"
            placeholder="Email"
            label="Email Address"
          />
          <Input
            type="password"
            placeholder="Password"
            label="Password"
          />
        </Card.Body>
        <Card.Footer>
          <Button variant="primary" size="md">
            Sign In
          </Button>
        </Card.Footer>
      </Card>
    </div>
  )
}
```

## TypeScript Support

All components are fully typed. Import types for better development experience:

```tsx
import { Button, type ButtonProps } from '@componentui/ui'

const CustomButton: React.FC<ButtonProps> = (props) => {
  return <Button {...props} />
}
```

## Dark Mode

Enable dark mode by adding the `dark` class to your root element:

```tsx
// Toggle dark mode
function App() {
  const [isDark, setIsDark] = useState(false)

  useEffect(() => {
    if (isDark) {
      document.documentElement.classList.add('dark')
    } else {
      document.documentElement.classList.remove('dark')
    }
  }, [isDark])

  return (
    <div>
      <button onClick={() => setIsDark(!isDark)}>
        Toggle Dark Mode
      </button>
      {/* Your components */}
    </div>
  )
}
```

## Customization

### Using className

Add custom styles using the `className` prop:

```tsx
<Button
  variant="primary"
  className="mt-4 shadow-lg"
>
  Custom Button
</Button>
```

### Using Tailwind Utilities

All components support Tailwind CSS utility classes:

```tsx
<Input
  label="Name"
  className="w-full max-w-md"
/>
```

## Best Practices

### 1. Accessibility

All components are designed with accessibility in mind:
- Proper ARIA attributes
- Keyboard navigation support
- Screen reader friendly

```tsx
<Button aria-label="Close dialog" onClick={handleClose}>
  √ó
</Button>
```

### 2. Performance

Import only what you need for better tree-shaking:

```tsx
// ‚úÖ Good - Named imports
import { Button, Input } from '@componentui/ui'

// ‚ùå Avoid - Default import
import ComponentUI from '@componentui/ui'
```

### 3. Form Handling

Use controlled components for forms:

```tsx
function MyForm() {
  const [email, setEmail] = useState('')

  return (
    <Input
      type="email"
      value={email}
      onChange={(e) => setEmail(e.target.value)}
      label="Email"
    />
  )
}
```
