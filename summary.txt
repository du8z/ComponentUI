🚀 QUICK START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install dependencies:
   $ pnpm install
   
   Or run the install script:
   $ ./install.sh

2. Start Storybook (recommended):
   $ pnpm dev:docs
   → Opens http://localhost:6006

3. Start development server:
   $ pnpm dev

4. Run tests:
   $ pnpm test
   $ pnpm test:coverage
   $ pnpm test:watch

Development:
  pnpm dev              - Start UI dev server
  pnpm dev:docs         - Start Storybook

Build:
  pnpm build            - Build component library
  pnpm build:docs       - Build Storybook static

Testing:
  pnpm test             - Run all tests
  pnpm test:coverage    - Run with coverage report
  pnpm test:watch       - Watch mode

Quality:
  pnpm lint             - Lint all packages
  pnpm format           - Format with Prettier
  pnpm format:check     - Check formatting

Maintenance:
  pnpm clean            - Clean all build artifacts

📝 CREATING COMPONENTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Each component needs:
  ✓ ComponentName.tsx        - Implementation
  ✓ ComponentName.test.tsx   - Unit tests
  ✓ ComponentName.stories.tsx - Storybook stories
  ✓ index.ts                 - Exports

Example:
  packages/ui/src/components/Button/
  ├── Button.tsx
  ├── Button.test.tsx
  ├── Button.stories.tsx
  └── index.ts

Don't forget to export in packages/ui/src/index.ts!
